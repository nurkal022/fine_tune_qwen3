# Датасет юридических вопросов и ответов (Казахстан)

## Описание

Датасет содержит **27,070 пар** вопросов и ответов по законодательству Республики Казахстан, собранных из открытых юридических консультационных порталов. Предназначен для fine-tuning языковых моделей (LLM) в области казахстанского права.

## Источники данных

### 1. dogovor24.kz
- **URL:** https://dogovor24.kz/questions
- **Описание:** Крупнейший казахстанский портал юридических консультаций
- **Тематика:** Широкий спектр юридических вопросов (семейное, трудовое, гражданское право и др.)
- **Особенности:** Подробные ответы профессиональных юристов

### 2. jurhelp.prg.kz
- **URL:** https://jurhelp.prg.kz
- **Описание:** Юридическая помощь онлайн
- **Тематика:** Общие юридические консультации
- **Особенности:** Краткие и структурированные ответы

### 3. yur.kz
- **URL:** https://yur.kz/questions
- **Описание:** Сервис поиска юристов в Казахстане с разделом Q&A
- **Тематика:** Семейные споры, земельное право, автоюрист, уголовные дела, наследство
- **Особенности:** Ответы с рекомендациями по поиску специалистов

### 4. pravosite.kz
- **URL:** https://pravosite.kz/questions
- **Описание:** Юридический онлайн-сервис с образцами документов
- **Тематика:** Трудовое право, договорные отношения, корпоративное право
- **Особенности:** Развёрнутые ответы со ссылками на НПА

## Формат данных

### training_data.jsonl (52 MB)
JSONL формат для fine-tuning. Каждая строка — JSON объект:

```json
{
  "instruction": "Ответь на юридический вопрос по законодательству Казахстана.",
  "input": "Текст вопроса пользователя",
  "output": "Текст ответа юриста",
  "metadata": {
    "id": "уникальный_id",
    "url": "ссылка_на_источник",
    "title": "заголовок_вопроса",
    "date": "дата_публикации",
    "category": "категория",
    "source": "источник_данных"
  }
}
```

### all_data.json (54 MB)
Полный JSON массив всех записей (для анализа и обработки).

### training_data.txt (47 MB)
Читаемый текстовый формат для просмотра данных.

## Тематические категории

Основные категории вопросов:
- Семейное право (алименты, развод, раздел имущества)
- Трудовое право (увольнение, отпуск, зарплата)
- Гражданское право (договоры, долги, взыскание)
- Жилищное право (аренда, купля-продажа, ЖКХ)
- Земельное право (участки, приватизация)
- Наследственное право
- Административное право
- Уголовное право
- Банкротство
- Исполнительное производство

## Использование

### Загрузка для обучения (Python)
```python
import json

# JSONL
data = []
with open('training_data.jsonl', 'r', encoding='utf-8') as f:
    for line in f:
        data.append(json.loads(line))

# Для Hugging Face datasets
from datasets import load_dataset
dataset = load_dataset('json', data_files='training_data.jsonl')
```

### Фильтрация по источнику
```python
dogovor24_data = [d for d in data if d['metadata']['source'] == 'dogovor24']
```
